**POST**

- Send a POST request to the following endpoint:

  ```bash
  GET /content/subscribers/add
  ```

  **Headers**

```bash
 Authorization:

 Key     x-api-key

 Value 1c468fa0696213bb17aa3bd0fa98f902522d16da0f9bb95f4582101e1677e74a
```

## Code

```js
   const axios = require("axios");

   const apiUrl = "{{BASE_URL}}/subscribers/add";
   const apiKey = "YOUR_API_KEY;

axios.post(apiUrl, { key: 'value' }, {
  headers: {
    'x-api-key': apiKey,
  },
})
  .then(response => {
    console.log('Data:', response.data);
  })
  .catch(error => {
    console.error('Error:', error.response ? error.response.data : error.message);
  });

```

**GET**

- To retrieve a list of Subscribers, send a GET request to the following endpoint:

  ```bash
  GET / content / subscribers;
  ```

**Headers**

```bash
 Authorization:

 Key     x-api-key

 Value 1c468fa0696213bb17aa3bd0fa98f902522d16da0f9bb95f4582101e1677e74a
```

## Code

```js
   const axios = require("axios");

   const apiUrl = "{{BASE_URL}}/subscribers";
   const apiKey = "YOUR_API_KEY;

   axios
     .get(apiUrl, {
       headers: {
         "x-api-key": apiKey,
       },
     })
     .then((response) => {
       console.log("Data:", response.data);
     })
     .catch((error) => {
       console.error(
         "Error:",
         error.response ? error.response.data : error.message
       );
     });
```
